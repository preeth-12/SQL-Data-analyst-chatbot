# SQL Data Analyst Chatbot

An AI chatbot that analyzes CSV files, generates SQL queries from natural language questions, and returns results instantly.

---

## 🚀 Features
- Upload and explore CSV datasets
- Ask questions in plain English
- Automatically generate SQL queries
- View query results in tabular format
- Extendable with LLMs (LangChain, Groq API)

---

## 📦 Installation

Clone the repository:
```bash
git clone https://github.com/preeth-12/SQL-Data-analyst-chatbot.git
